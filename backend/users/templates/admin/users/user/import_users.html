Valid Roles

  • AG - Additional General (head of office, no section)
  • DAG - Deputy Additional General (section head)
  • SrAO - Senior Accounts Officer
  • AAO - Assistant Accounts Officer
  • auditor - Auditor
  • clerk - Clerk

Notes

  • Missing sections/subsections are auto-created from import values
  • AG users typically have no section/subsection (leave those columns empty)
  • Existing users (same username) will be skipped
  • Imports now run in the background; use Refresh Status to monitor progress
  • Passwords are stored securely (hashed)
